# Cześć, jestem Przecinek 👋

Zajmuję się tworzeniem zaawansowanych **pluginów**, **paczek** oraz **gotowych sieci** dla serwerów **Minecraft**. 🚀

---

## Specjalizacja

* 🛠️ **Programowanie autorskich pluginów** pod silniki Spigot oraz PaperMC
* ⚡ **Maksymalizacja wydajności** i kompleksowa konfiguracja środowisk Minecraft
* ⚙️ **Automatyzacja procesów**, tworzenie backendu oraz narzędzi pomocniczych
* 🌐 **Architektura i wdrażanie** rozbudowanych, wieloserwerowych sieci (Proxy)
